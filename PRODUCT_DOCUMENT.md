# Product Document: Credit Card Rewards Strategist

## Executive Summary

**Product Name:** Credit Card Rewards Strategist (SaveSage-style)  
**Platform:** ChatGPT (via Model Context Protocol - MCP)  
**Version:** MVP 1.0  
**Last Updated:** 2024

### What We're Building

A conversational AI-powered credit card rewards optimization assistant that runs inside ChatGPT. The product helps users maximize credit card rewards by providing real-time recommendations on which card to use for each purchase, when to redeem points, and which new cards to add to their portfolio.

### Core Value Proposition

**"Which card should I use, when should I redeem, and why?"** - Answered instantly through natural conversation in ChatGPT.

---

## 1. Product Vision & Mission

### Vision Statement

To become the most trusted AI-powered credit card rewards strategist, helping millions of users unlock maximum value from their credit cards through intelligent, personalized recommendations delivered in natural conversation.

### Mission Statement

Empower credit card users to make optimal spending decisions by providing real-time, data-driven recommendations that maximize rewards value, surface relevant deals, and optimize their entire credit card portfolio‚Äîall through an intuitive conversational interface.

### Core Principles

1. **Conversational First**: No dashboards, no complex UIs‚Äîjust natural conversation
2. **Trust & Security**: Never asks for sensitive data (card numbers, CVV, OTPs)
3. **Data-Driven**: All recommendations based on factual card data and reward rules
4. **User-Centric**: Personalized to each user's card portfolio and spending patterns
5. **Transparent**: Clear explanations of why recommendations are made

## 3. Market Opportunity & Problem Statement

### The Problem

Credit card users face several challenges:

1. **Decision Fatigue**: With multiple cards, users struggle to remember which card offers the best rewards for each category
2. **Information Overload**: Reward structures are complex‚Äîdifferent rates for different categories, caps, expiration dates
3. **Missed Opportunities**: Users leave money on the table by:
   - Using the wrong card for purchases
   - Not tracking reward caps
   - Missing card-specific deals and offers
   - Not optimizing their portfolio
4. **Lack of Personalization**: Generic advice doesn't account for individual card portfolios
5. **Time-Consuming Research**: Users spend hours researching which card to use or which new card to get

### Market Opportunity

**The Credit Card Rewards Market:**
- US consumers earn ~$50 billion in credit card rewards annually
- Average user earns ~2% back on purchases
- Optimized users can earn 5-15% back through strategic card usage
- **Opportunity Gap:** $25-65 billion in unrealized rewards value

**The ChatGPT Ecosystem:**
- ChatGPT has 200M+ monthly active users globally
- Growing adoption of MCP (Model Context Protocol) tools
- Users increasingly expect AI assistants for complex tasks
- **Opportunity:** First-mover advantage in credit card optimization via ChatGPT

### Competitive Landscape

#### Direct Competitors

## 6. Key Features (Detailed)

### 6.1 Card Matching & Spend Guidance

**What It Does:**
Answers "Which card should I use for $X on [category/merchant]?"

**How It Works:**
1. User asks: "Which card for $200 groceries?"
2. System compares all active cards
3. Calculates rewards for each card
4. Normalizes points/miles/cashback to dollar value
5. Recommends best card with explanation

**Key Capabilities:**
- Category-based matching (travel, dining, groceries, gas, shopping, online, general)
- Merchant name recognition (Whole Foods ‚Üí groceries, Uber ‚Üí travel)
- Multi-card comparison
- Trade-off explanations when cards are close (within 10% value)
- Cap awareness (warns when approaching limits)

**Value Delivered:**
- Users earn 2-3x more rewards through optimized card usage
- Saves time (instant answers vs. manual research)
- Prevents missed opportunities (cap warnings)

### 6.2 Personalized Card Recommendations

**What It Does:**
Suggests new credit cards based on user's portfolio and spending patterns.

**How It Works:**
1. Analyzes user's existing cards
2. Identifies gaps (categories with low/no coverage)
3. Matches user preference (travel/cashback/balanced)
4. Recommends cards with highest value potential
5. Shows annual fees, benefits, target audience

**Key Capabilities:**
- Gap analysis (missing high-return categories)
- Overlap detection (redundant cards)
- Value estimation (annual rewards - annual fee)
- Preference matching
- Filters out cards user already has

**Value Delivered:**
- Users discover cards that complement their portfolio
- Estimated value increase per recommended card
- Reduces research time for new card applications

### 6.3 Deal Surfacing & Alerts

**What It Does:**
Surfaces active deals and offers relevant to user's cards and spending.

**How It Works:**
1. Maintains database of active deals
2. Matches deals to user's cards
3. Prioritizes by category and spending patterns
4. Displays in carousel widget
5. Filters by category or card

**Key Capabilities:**
- Card-specific deals
- Category filtering
- Validity tracking
- Discount percentage/amount display
- Merchant information
- Terms and conditions

**Value Delivered:**
- Users discover deals they might miss
- Stack deals on top of card rewards
- Increased savings per purchase

### 6.4 Rewards Tracking

**What It Does:**
Tracks points, miles, and cashback balances across all cards.

**How It Works:**
1. User updates balances (manual entry)
2. System tracks expiry dates
3. Warns when rewards expire soon (90 days)
4. Provides summary across all cards

**Key Capabilities:**
- Multi-card balance tracking
- Expiry date warnings
- Reward type summary (points/miles/cashback)
- Last updated timestamps
- Notes per card

**Value Delivered:**
- Prevents point expiration
- Centralized view of all rewards
- Helps plan redemptions

### 6.5 Wallet Summary & Analysis

**What It Does:**
Provides overview of user's credit card portfolio with strengths and weaknesses.

**How It Works:**
1. Analyzes all active cards
2. Identifies best categories per card
3. Highlights strong coverage areas
4. Points out weak spots (categories with low/no coverage)
5. Suggests improvements

**Key Capabilities:**
- Portfolio overview
- Strengths identification
- Weak spots detection
- Category coverage map
- Optimization suggestions

**Value Delivered:**
- Users understand their portfolio better
- Identifies optimization opportunities
- Guides portfolio expansion

### 6.6 Merchant Category Lookup

**What It Does:**
Automatically determines spending category for merchant names.

**How It Works:**
1. User mentions merchant: "Buying from Whole Foods"
2. System looks up merchant in database
3. Returns suggested category (groceries)
4. If confidence is low, asks user to confirm
5. Proceeds with card recommendation

**Key Capabilities:**
- Built-in merchant mappings (20+ common merchants)
- Custom merchant mappings
- Confidence scoring
- Category suggestions

**Value Delivered:**
- Reduces user friction (no need to specify category)
- More accurate recommendations
- Better user experience

### 6.7 Enhanced Chat (AI-Powered)

**What It Does:**
Answers complex questions about cards, rewards, and strategy using AI.

**How It Works:**
1. User asks complex question: "Do I have lounge access on any card?"
2. System uses OpenAI API with user's card data as context
3. Returns intelligent, personalized answer
4. Explains reasoning

**Key Capabilities:**
- Complex question answering
- Portfolio-aware responses
- Strategy explanations
- Redemption guidance
- Benefit inquiries

**Value Delivered:**
- Users get answers to nuanced questions
- Deeper understanding of card benefits
- Strategic guidance beyond simple recommendations
